一、js创建对象、扩展属性

使用花括号直接创建

var obj = {
    id:"011",
    name:"我自己",
    test:function(){
        //方法体
    }
}

//对象扩展属性，扩展属性和方法只有当前扩展对象才有，其他没有
obj.age = 21;

用构造函数直接创建

//定义构造函数
function Student(num, name){
    this.stuNum = num;
    this.stuName = name;
    this.funName = function(a){
        console.log(a);
    }
    this.funName2 = myMethod;
}

//外部定义方法
function myMethod(){
    //方法体
}

var s = new Student("101", "张三");

//构造器扩展属性，所有属性都有
Student.prototype.stuGender = "男";

二、内置对象

2.1 String对象

String方法或属性 说明
length 获取字符串的长度
charAt( index ) 获取索引为index的字符
indexOf( str ) 获取第一次出现str的位置
lastIndexOf( str ) 最后一次
split( char ) 以char分割字符串，存放在数组里
substring( index1, index2 ) 两个参数，截取index1到index2的字符串（左闭右开）
substr( index, num ) 从index开始截取num个字符串
toUpperCase() 转大写
toLowerCase() 转小写
match( reg ) reg正则表达式，匹配返回字符串，否则null

2.2 Math对象

直接调用Math.属性/方法

方法 说明
random() 产生 0~1 小数
round(num) 四舍五入
floor(num) 获取比参数小的最大整数
ceil(num) 获取比参数大于等于的最小整数

2.3 Array对象

数组中可以放置不同数据类型

方法 说明
join(char) 数组通过char连接成字符串
slice(index1, index2) 截取数组中index1到index2之间的元素（左闭右开）
reverse() 反转数组
