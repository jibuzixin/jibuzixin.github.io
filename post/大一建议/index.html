  • 算法 刷题网站：力扣算法刷题 推荐书：《深入浅出程序设计竞赛 基础篇》洛谷的算法书还不错，c++写的，基本算法都用c++写，有个stl标准库函数能直接用，很方便 还有一本比较深入一点，如果你要我过年给你带回去，我都买了 **建议：**就是刷题，不会就百度，然后再刷题，再百度

  • markdown Markdown 基本语法 | Markdown 官方教程 Markdown 语法速查表 | Markdown 官方教程 软件等你有电脑给你发

  • linux **视频：**B站一大把，学个基础就行，先不必深入 推荐书籍：《linux就该这么学》看到第九章就可以了

  • makefile 推荐网址：GCC makefile我也没学，GCC看了就行了，这些够你学了

一、大一精简路线

路线是编程的路线，学校的东西自己要看着取舍，剩下的都是学计算机通用的

一些计算机基础（计算机组成，操作系统，计算机网络等等专业课）一定要好好学，学的是思想，记不记得以后也是需要了再百度

学会自学（我是烂了点），多学多看，代码多敲（一定要）

不会问我，我那会是打算一直学c/c++的，但最后面对现实，因为找工作嘛，学java后端开发去了，c/c++就没学了，但也还是可以的，起码够你学到大二了

先学c语言

c语言==初学==不需要 那么深入，因为 要先快点入门，一些基本的语法会用就可以了，可能刚开始有些东西记不住（你英语不错，应该不是问题），那还是敲代码少。

大一敲代码一定要多，所以在基本语法学完之后去看==算法==

之后是算法

算法大一没什么重要事情，可以安安静静好好学学算法。刚开始不需要追求做题的数量，两三天一道算法题就可以了。

例如一些基本的要会：

  • 红黑树，二叉树，平衡二叉树
  • 最大最小堆
  • 堆，栈
  • 链表，也就是要用c语言的指针
  • 等等。。。概念问题，像什么线段树，动态规划，复杂度。。。。

以上学算法的时候都会遇到，顺势而为

以上就是上学期就够了

剩下的多看看，多听听，把握住度，因为计算机范围挺广，如果找不到自己要学的方向，你只能浪费大一这一年的时间。要做到广学，对计算机有一个体系的概念，然后 找到方向，再深入学一门精通。

二、必学扩展（有时间学，值得一试）

一定要花时间，c语言学了，不会浪费你的时间，对你以后的编程受益匪浅，好好学。

一下就是有时间学学，但花的时间也不多，尽可能看看

这学期，时间不多，就学个c语言，算法够了

第一个就是 markdown

如果想要做笔记，markdown对计算机学习来说是一个不错的选择，方便，快捷，占用小，适用面广。

==提醒：==我现在给你写的这个就是用markdown写的

第二个是linux基本指令

如果你想要进阶，对计算机有些理解，学习一些linux相关对知识对之后的c语言有很大帮助，况且如果你要做嵌入式，或者一些c语言的开发，没有集成的编译工具你很难编译多文件任务

第三个就是makefile

在学习linux之后，再去看一些用指令编译c语言文件时，更得心应手。其实这两个颠过来学也可以
