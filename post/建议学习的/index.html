建议学习的

给我的发小总结的一些大一的东西

Posted by 忌不自信 on Tuesday, October 25, 2022

一、夯实基础

学院大部分的都会学java后端和前端开发，不过具体的还是要根据自己的兴趣来。刚开始的话把c语言学好对以后别的语言学习很有帮助，如果能理解c的指针会更好。再看一些算法和数据结构的题，学习数据库中的一个，夯实基础就行。

不排除有些人对c语言不感冒，如果实在学不懂也不要怕，可以试试其他语言，比c可能会简单一些。例如：java、python、c#等等

1.0、要学的

  • c语言
  • java基础
  • MySQL数据库（往后要用java去操作数据库）
  • 算法（刷题看书）
  • linux基本指令（早些接触）
  • vim编辑器（早些接触）
  • Shell编程（早些接触）
  • makefile（早些接触）
  • markdown（早些接触）（这个就是用markdown写的，做笔记）

1.1、刚开始基础

  • c语言： c语言学好对学其他语言很快，剩下的就是思想的问题，语言只是工具

  • 算法和数据结构：

    掌握数据结构和常见的算法是最基本的，这也是大一可以早些打基础的，虽然说以后的课程会学到，但如果能把算法坚持下来，参加一些算法竞赛，去学校的acm实验室打打比赛获得一些奖项，对找工作面试帮助很大，不可小觑

  • 数据库：

    • 学院学的都是MySQL数据库，B站也有资源

    这也是基本的，掌握一门数据库也是一样，再接触新的数据库的时候只是语法的不同，大体思想还是一样

相关的软件不知道的可以找我

1.2、java后端开发的学习

大一开始就挺好，早点开始早点结束，放弃c语言也是可以的，就看自己的学习能力了。

前端后端学院都会学教，但是具体也要精通一门，学后端的相比较前端难一点，但都大差不差，但后端工资一般来说一个水平的要比前端高。

学后端的要能看懂前端的，所以前端不能马虎，多多少少也要懂一些，同样的B站有很多资源

==建议：==模棱两可这个可以直接去B站搜javaweb视频看，一遍不行两遍三遍，然后跟着他的路线往后学习就行

  • 前端：

    • html、css、JavaScript 这三个必学，可以看菜鸟教程，也可以去B站看视频，很快就可以入门，然后跟着做做就行
    • 然后就是一些前端框架，Vue、node.js、ajax，这些我也不太会，懂一点就行，但如果要学前端的可以自己摸索摸索

  • 后端：

    • java基础 这个当然是关键，面向对象编程、反射、函数式接口、线程、并发等等，刚开始会点基础就行，边学边用，更深的我推荐两本书《java核心技术卷1》和《java核心技术卷2》
    • 然后就是B站黑马的这个路线Java学习路线图

    这个没得说，java人太多，网上都是资源，一找一大把，刚开始我还是建议打基础，基础很重要，什么技术，什么语言，都是工具，重要的是思想。

之后就是缝缝补补，学到哪哪不会看哪

1.3、计算机基础

这个没什么说的，就是看书看视频，慢慢的有个体系，如果之前有学过计算机的更简单了

  • 计算机组成原理
  • 计算机网络
  • 操作系统
  • 数据结构
  • 设计模式
  • linux基本指令（与其相关内的Vim编辑器，Shell编程）

剩下的就是看自己感兴趣的地方玩玩看看，总会有收获

二、推荐资源

==推荐：==

网址、资源其实可以百度都有，我顺便把我看的，给你放到前面这，具体怎么学看你，一下子这些东西可能也学不完，也不先学精，广学找到合适自己的方向再去深学更好

  • c语言 视频：翁凯的c语言 书籍推荐： 学校的就可以（也就是最最基础的） 进阶书籍：《 c prime 》（应该是红皮的，新版不知道），其实看 《c++ prime 》也可以 不必入深也行
  • 算法 刷题网站：力扣算法刷题 推荐书：《深入浅出程序设计竞赛 基础篇》洛谷的算法书还不错、适合入门，c++写的，有个stl标准库函数能直接用，还有个难一点的书，忘了叫啥，需要的话我可以借给你 **建议：**就是刷题，不会就百度，然后再刷题，再百度
  • markdown Markdown 基本语法 | Markdown 官方教程 Markdown 语法速查表 | Markdown 官方教程 软件在分享的链接里typora-setup-x64.exe
  • linux **视频：**B站一大把，学个基础就行，先不必深入 推荐书籍：《linux就该这么学》看到第九章就可以了
  • makefile 推荐网址：GCC makefile我也只学了一点跟我一起写makefile，GCC看了就行了，这些够你学了
